Primeiro passo -> 
importar o repositório fonte criando um publico de nome semelhante : https://github.com/filipedeschamps/ideia-unica

Segundo passo ->
após importação, via linha de comando na pasta de usuário(pwd), digitar :  git clone https://github.com/madtbr/ideia-unica.git

Terceiro passo ->
tendo previamente instalados o NODE(10.3 ou superior) e o NPM, nesse projeto em específico, entrar com os comandos :

npm init -y pra criar o arquivo package.json com configs padrão
npm install next react react-dom

Quarto passo ->
uma observação importante, é que o comando export indica a funcao que vai se tornar uma página
pra testar e rodar o projeto : npm run dev
